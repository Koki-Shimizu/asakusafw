========
WindGate
========
WindGateはThunderGateと同様にバッチに対するデータの外部入出力を行うモジュールです。

ThunderGateに対して、WindGateは「ポータブルなThunderGate」して位置づけられ、以下の特徴を持ちます。

様々なRDBに対応
~~~~~~~~~~~~~~~
DBMS固有の機能に依存せず、標準SQL/JDBCインターフェースのみを使用した実装を提供します。

様々なOSに対応
~~~~~~~~~~~~~~
Linux以外のOS (Windows Server等)にデプロイして使用することを想定し、Unixコマンド(SSH等)への依存を排除しています。

業務テーブルを変更不要
~~~~~~~~~~~~~~~~~~~~~~
処理対象テーブルに対するThunderGateの管理カラムの追加が不要です。ただしThunderGateが提供していたロック機構などの仕組みが一部提供されません。

外部から拡張しやすい
~~~~~~~~~~~~~~~~~~~~~
データ入出力部分を拡張しやすいアーキテクチャを採用しています。
